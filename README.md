> Springboot를 활용한 게시판 프로젝트 
> (현재 local 동작 상태, EC2 세팅 완료, RDS 연결 중)

## Version
* Java 8.0
* Springboot 2.1.7
* JUnit4
* mustache

## Skills
* JUnit : 테스트 자동화
* Lombok : 코드 간략 표현
* Dirty Checking : Update시 쿼리를 날리지 않음
* JPA Auditing : 생성 및 수정 시간 관리
* OAuth 2.0 : 구글 및 네이버 로그인

## Features
* 게시판 기능 (Posts)
  * 게시글 조회
  * 게시글 등록
  * 게시글 수정
  * 게시글 삭제
  
* 회원 기능 (Users)
  * 구글 / 네이버 로그인
  * 글 작성 권한
  * 본인 작성 글에 대한 수정 권한
